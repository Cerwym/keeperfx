## Quick Reference Commands

```bash
# Build everything from scratch
make clean && make standard && make heavylog && make package && make installer

# Just build the installer (after package exists)
make installer

# Build only the 7z patch
make package

# Clean all outputs
make clean && make clean-package && make clean-installer

# GitHub Actions will run automatically on push to main/develop
# Or trigger manually from Actions tab: "Build GOG Installer"
```
